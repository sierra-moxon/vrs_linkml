Class: Gene

A reference to a Gene as defined by an authority. For human genes, the use of hgnc as the gene authority is RECOMMENDED.

URI: GA4GHVRSDefinitions:Gene

Slots

Name Range Cardinality Description Info
type NONE 1..1 MUST be "Gene" .
gene_id CURIE 1..1 A CURIE reference to a Gene concept .
