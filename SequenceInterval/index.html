Class: SequenceInterval

A SequenceInterval represents a span on a Sequence. Positions are always represented by contiguous spans using interbase coordinates or coordinate ranges.

URI: GA4GHVRSDefinitions:SequenceInterval

Slots

Name Range Cardinality Description Info
type NONE 1..1 MUST be "SequenceInterval" .
start integer 1..1 The start coordinate or range of the interval. The minimum value of this coordinate or range is 0. MUST represent a coordinate or range less than the value of end. .
end integer 1..1 The end coordinate or range of the interval. The minimum value of this coordinate or range is 0. MUST represent a coordinate or range greater than the value of start. .
